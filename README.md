**# ☕ Java Canteen Ordering System

A simple, console-based Java application that simulates a Canteen (Cafeteria) ordering and checkout system. This project demonstrates core Object-Oriented Programming (OOP) concepts such as classes, encapsulation, and input validation.

## ✨ Features
* **Interactive Menu:** Users can choose from a list of products (Toast, Coffee, Juice, Water, Bagel).
* **Order Cancellation:** Ability to cancel the most recently added item.
* **Smart Discounts:** * 10% discount for orders over 15€.
  * 5% discount for ordering 8 or more items.
  * 5% discount for students.
  * *Note: Maximum total discount is capped at 15%.*
* **Payment Validation:** Ensures the user enters a valid and sufficient payment amount, calculating the proper change.
* **Detailed Receipt:** Prints a neatly formatted receipt at the end of the transaction.
